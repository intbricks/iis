## AWS S3 Connector

### Examples 
```
connection "aws" "s3" {
  access_key = env("ACCESS_KEY")
  secret_key = env("SECRET_KEY")
  region = "us-east-1"
}

connector "s3" "download_s3" {
  bucket     = "lab-storage"
  key        = "VIM-quick-ref.pdf"
  operation  = "GetObject"
  connection = aws.s3
}

connector "s3" "upload_s3" {

  key       = "sftp2s3.pdf"
  operation = "PutObject"
  bucket    = "lab-storage"
  
  connection = aws.s32sftp
  
  tags = {
    Project           = "S32SFTP Integration Sample"
    "user:CostCenter" = "S32SFTP Integration"
    Event             = "upload to S3"
  }
}
```

### Parameters

| Parameter Name                                       | Type           | Required? | Default Value  |
|------------------------------------------------------|----------------|-----------|----------------|
| `bucket`: S3 bucket name                             | String         | Yes       ||
| `key`: S3 resource key                               | String         | Yes       ||
| `operation`: S3 operation (`GetObject`, `PutObject`) | String         | Yes       ||
| `connection`: AWS connection reference. See [connection docs](conn.md).             | AWS Connection | Yes       ||
| `tags`: Specify meta data using (key,value)          | Map            | No        ||
